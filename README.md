# Weather App (API-based Application)

This project is created for assignment submission.

## Features

- Uses external Weather API
- Fetches live weather data using HTTP request
- Search weather by city name
- Displays city name
- Displays temperature
- Displays weather condition
- Dynamic UI updates after search
- Handles loading state
- Handles error state

## Technologies Used

- Flutter
- HTTP Package
- JSON API Response

## Widgets Used

- Scaffold
- AppBar
- TextField
- IconButton
- Column
- CircularProgressIndicator
- Text
